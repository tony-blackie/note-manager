### Install:
```
npm install
npm install babel-preset-react
npm install babel-preset-es2015
```

### Run:
```
npm start
```

### Tests:
##### All tests:
```
npm test
```
##### Single test:
```
npm test -- FileName.spec.js
```
##### Coverage report:
```
npm run coverage
```

### About
```
Built with: Webpack
Front-end: React, Redux, ES6, Babel
Back-end: Express, mySQL
Tests: Jest, Enzyme
```
