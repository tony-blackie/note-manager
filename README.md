### Install:
```
npm install
npm install babel-preset-react
npm install babel-preset-es2015
```

### Run:
```
npm start
Go to localhost:8080
```

### Tests:
##### All tests:
```
npm test
```
##### Single test:
```
npm test -- FileName.spec.js
```
##### Coverage report:
```
npm run coverage
```

### About
```
Front-end: Typescript, React, Redux, ES6, Babel, Webpack
Back-end: Python, Django, SQLite
Tests: Jest, Enzyme
```
